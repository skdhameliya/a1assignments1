JavaScript Programming
Modal title
Course Description

This course will introduce client-side browser scripting using JavaScript programming techniques. Students will focus on client-side web scripting and develop their ability to manage interactive websites using JavaScript Programming while manipulating a page's HTML and CSS code. Students will work with JavaScript concepts to handle both simple and complex client-side webpage events. This course will also introduce code version control and code collaboration. Accessibility features of JavaScript coding and debugging best practices will also be explored.

Course Outcomes

Successful completion of this course will enable the student to:
Describe the basic structure of JavaScript and best practices for the creation of maintainable JavaScript code.
Create efficient JavaScript functions and methods using objects, events, media, plugins, and widgets to enhance a web site based on industry best practices.
Validate website data using JavaScript to ensure data integrity and reliability.
Use industry-standard tools and methodologies to troubleshoot and debug JavaScript code.
Use jQuery to integrate plugins and UI widgets into a website to enhance site functionality.

Assignment 1 - Convert number grades to letter grades

Create an application that asks the user to enter a number grade between 0 and 100. The application should get the number grade value, convert it into letter grade, then displays the letter grade result. The application should also validate the entered value to be between 0 and 100, otherwise, it should display an error message. Finally, the application should ask the user if they wish to exit the application.

Assignment 2 - Class grade stats

Create an application that allows the user to add student grades. The application should also calculate and display different statistics related to the grades entered. Refer to the following images for reference.

Assignment 3 - Working with timers

In this assignment, you’ll create an application that displays the current time in hours, minutes, and seconds. The display should use a 12-hour clock and indicate whether it’s AM or PM.

Assignment 4 - Working with a carousel

In this assignment, you’ll update a carousel application by adding animation and changing its behavior.

Assignment 5 - Working with form validation

In this assignment, you’ll use JavaScript to validate a reservation request form.
